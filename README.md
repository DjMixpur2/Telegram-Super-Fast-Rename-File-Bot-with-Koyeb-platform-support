# Telegram Super Fast Rename File Bot with Watermark and Thumbnail

A Telegram bot that allows users to rename files, add watermarks to images and videos, create custom thumbnails, and save them as media or file.

## Features:
- Rename files quickly.
- Option to save as "media" or "file".
- Add a watermark to images and videos at the bottom-left corner.
- Custom thumbnail for each media.
- Koyeb hosting support.
